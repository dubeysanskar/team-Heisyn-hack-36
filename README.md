## ğŸ”’ Security & Privacy (Aligned with Hack36 9.0 Secure)

Security is deeply embedded in every layer:
- **Data Encryption:** AES-256 at rest; TLS 1.3 in transit.
- **Authentication:** OAuth2 / OpenID Connect with MFA.
- **Authorization:** RBAC + Attribute-based access control.
- **n8n Hardening:** Encrypted credentials, isolated workflows.
- **LLM Safety:** Prompt filtering, hallucination prevention, output guardrails.
- **Audit Logging:** Immutable event trails for critical actions.
- **Consent Management:** Transparent data usage agreements with export/delete options.
- **Incident Response:** SIEM-based monitoring and anomaly alerts.

## ğŸ§© n8n Pipeline Example (Recruitment Flow)

1. **Trigger:** Job posted via ATS/webhook.  
2. **Fetch Candidates:** Resume parsing and normalization.  
3. **Pre-screen:** Filter by CGPA, year, skills, location.  
4. **LLM Scoring:** Evaluate profiles using skill-match models.  
5. **Scheduling:** Auto-schedule AI interviews.  
6. **AI Interview Session:** Evaluate answers in real-time.  
7. **Report Generation:** Produce ranked candidate list with evidence.  
8. **HR Notification:** Send final reports to dashboard.
